# 歌词格式示例

以下是角色主题曲/OP/ED 歌词的标准格式。歌词使用段落标记 `[section]`，与 Music API 的歌词输入格式兼容。

```markdown
# 一念之间

角色：若曦
情感：穿越时空的思念与遗憾
视角：第一人称

---

[intro]
(钢琴前奏)

[verse]
月光洒在宫墙上
我数着影子的方向
三百年前的雪还没融化
而我已经走了太远太远

[verse]
你转身时风很轻
像那年递来的暖手炉
我知道结局写在史书里
却还是握紧了你的手

[pre-chorus]
如果时间是一道门
我愿意再推开一次

[chorus]
一念之间 跨越百年
只为在你眼中 停留一瞬
一念之间 山河变迁
我带不走什么 只带走思念

[verse]
宫廷的钟声敲了又敲
每一声都在催我回去
可我的心 留在了
那个雪夜 那盏灯

[chorus]
一念之间 跨越百年
只为在你眼中 停留一瞬
一念之间 山河变迁
我带不走什么 只带走思念

[bridge]
也许在某个平行的时空
我们不必说再见
也许在那里
爱不需要穿越

[chorus]
一念之间 跨越百年
只为在你眼中 停留一瞬
一念之间 天地之间
这份爱不会散 化作永远

[outro]
(古筝渐弱)
```

## 格式要点

- 一级标题为歌曲名
- 头部包含：角色、情感、视角（角色曲必须；OP/ED 可省略角色字段）
- 段落标记使用方括号：`[intro]`、`[verse]`、`[pre-chorus]`、`[chorus]`、`[bridge]`、`[outro]`
- 纯器乐段用括号注明乐器：`(钢琴前奏)`
- 每行歌词不超过 20 个字（适合演唱节奏）
- 标准结构推荐：intro → verse → verse → pre-chorus → chorus → verse → chorus → bridge → chorus → outro
- 歌词语言根据剧集文化背景选择（中文/英文/混合）
- 歌词中的意象应与角色的关键场景呼应
