# 画风分析示例

以下是一张参考图的画风分析报告示例，展示分析输出的格式和深度。

---

参考图：一张二次元角色图——蓝发少女穿校服，坐在窗边看书，阳光洒进来，整体明亮柔和。

---

## 画风溯源

| 项目 | 结果 |
|------|------|
| 风格家族 | 日系动漫 (jp_anime) |
| 风格 ID | jp_cel（赛璃璐） |
| 置信度 | 0.85 |
| 判断依据 | ① 均匀平涂色块，无渐变上色 ② 清晰黑色描边，线条粗细一致 ③ 二分阴影，明暗分界锐利 |
| 年代感 | 10s-20s（现代新番风格）|
| 制作方式 | 数码赛璃璐（digital cel） |
| 第二可能 | jp_ln（轻小说插图）— 因为日常感强、色彩偏柔 |

---

## 10 维分析

| 维度 | 描述 | 显著度 |
|------|------|--------|
| 线条 | 均匀黑色描边，粗细一致，干净利落，无手绘笔触感 | ⭐⭐⭐ 0.9 |
| 上色 | 标准赛璃璃平涂，二分阴影，高光为白色硬边块 | ⭐⭐⭐ 0.9 |
| 人物比例 | 标准日漫比例（约 6-7 头身），不夸张也不写实 | ⭐⭐ 0.6 |
| 面部画法 | 大眼（占脸部 1/3）、小鼻（一点或阴影暗示）、小嘴、瞳孔有高光层 | ⭐⭐ 0.7 |
| 色彩体系 | 明亮温暖，以蓝（发色）+ 白（校服）+ 暖黄（阳光）为主，饱和度中等偏高 | ⭐⭐⭐ 0.8 |
| 光影处理 | 阳光从窗户打入的侧光，人物面部有明确的受光面/背光面，窗框投影 | ⭐⭐⭐ 0.85 |
| 背景风格 | 教室窗边场景，背景较详细（有桌椅、窗帘、树影），但精度低于人物 | ⭐⭐ 0.6 |
| 年代感 | 现代新番风格（10s-20s），数码上色痕迹，干净无噪点 | ⭐⭐ 0.5 |
| 媒介质感 | 纯数码，无纸张/笔触纹理 | ⭐ 0.3 |
| 文化圈 | 日系动漫，校园日常题材，典型"日常番"气质 | ⭐⭐⭐ 0.8 |

---

## 3 维坍缩

| 维度 | 得分 | 分析 |
|------|------|------|
| **画风** (线条+上色+媒介+年代) | **0.78** | 标准赛璃璃，干净数码感，无特殊技法 |
| **角色** (比例+面部+光影) | **0.72** | 标准日漫面部，光影是亮点（窗边光） |
| **调性** (色彩+背景+文化圈) | **0.73** | 日常校园的温暖感，蓝+暖黄的配色和谐 |

**灵魂轴**：画风（赛璃璃的平涂+描边是最强辨识度特征）
**灵魂一句话**：标准赛璃璃动画截图感 + 窗边暖光的日常氛围，"好像在看一部治愈日常番"。

---

## Remix 方向推荐

基于灵魂分析，保留画风（赛璃璃平涂）和调性（温暖日常），推荐：

1. **咖啡店看书** — 同样的"安静阅读"情绪，换到咖啡店场景，暖色灯光替代阳光
2. **屋顶吹风** — 放学后坐在天台，保留侧光打法，蓝天+校服的色彩搭配
3. **图书馆角落** — 更安静的氛围，书架围绕，柔和顶光，同样的专注表情
